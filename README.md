# GESTIÓN DE FINANZAS

## DESCRIPCIÓN
Este es un pequeño proyecto de una app para gestionar todos los gastos y ganancias de un usuario.

## TECNOLOGÍAS
Las tecnologías usadas son HTML5, CSS3 y JavaScript.
